Backup dos dados

Seus dados são salvos automaticamente neste navegador. Faça backups periódicos pra não perder se trocar de máquina ou limpar o cache.

Adicionar ano

Dinheiro em conta

Diga quanto você tem em conta hoje para calibrar a tela Início. A partir daqui o app soma o que sobra e desconta o que falta a cada mês — o saldo nunca zera na virada.

Backups automáticos

📲 Instalar app

Instale o FinanceApp na tela inicial do seu dispositivo para acesso rápido, igual ao Todoist.

Se o botão não aparecer, no Chrome use o menu ⋮ → "Adicionar à tela inicial".
No Safari (iPhone) use o botão compartilhar → "Adicionar à Tela de Início".

Zona de perigo

Apaga todos os dados salvos e restaura o app ao estado inicial. Não dá pra desfazer.
